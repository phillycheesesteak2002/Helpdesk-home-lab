# Helpdesk Home Lab – Zendesk Ticketing System

## Overview

This project simulates a real-world IT Helpdesk environment using Zendesk.

I created and resolved support tickets to practice troubleshooting and incident management.

## Skills Demonstrated

• Ticket creation and management  
• Password reset support  
• Microsoft Outlook troubleshooting  
• Printer troubleshooting  
• WiFi troubleshooting  
• Incident documentation  
• End-user support  

## Tools Used

Zendesk  
Windows 11  

## Example Tickets

• User cannot login to Windows computer  
• Microsoft Outlook not opening  
• Network printer not printing  
• User unable to connect to WiFi  

## Purpose

This lab demonstrates my ability to support users and resolve technical issues using a professional ticketing system.
